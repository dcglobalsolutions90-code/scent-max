Network Scanner

Idle
Browser-based scan — no MAC addresses, hostnames, or device names (iOS blocks that at the OS level for any web page). This only shows which IP addresses on your subnet appear to respond, using a crude image-load timing trick, since browsers can't do a real ping or ARP sweep. Expect some devices to be missed. Tapping a result opens that device's own web page in a new tab if it has one (routers, smart plugs, printers often do) — phones and laptops usually won't show anything.
